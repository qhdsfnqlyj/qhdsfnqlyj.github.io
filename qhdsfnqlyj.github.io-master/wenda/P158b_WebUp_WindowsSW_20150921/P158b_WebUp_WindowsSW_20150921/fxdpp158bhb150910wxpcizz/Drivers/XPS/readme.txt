Additional information of driver for XPS for Windows Vista(R),
Windows Server(R) 2008, Windows(R) 7

This driver for XPS is a printer driver for application software developed for
"Windows Vista" or later version of Microsoft(R) Windows.

It may not operate properly with application software for any Windows previous
to Windows Vista (including Office 2007) due to some issues arising out of
 Windows Vista compatibility problem of Microsoft.
